introduction-to-some-cmd-in-linux

Posted on 2020-07-15 | In 总结

因为实验室的webserver是Linux系统，而我日常使用win，初次接触还是有很多命令要学习，确实很令我头疼。

在这里，记录一下一个初学者的commad:

ls 列出当前目录下的文件 
    ls -l 是以列表形式的展示，简写： ll
    ls在windows的powershell中也支持，默认以列表展示（仅支持ls）

pwd 当前目录

cd target_path 与windows一致，切换路径

cat filename 查看文件的全部内容
    head -n filename 查看文件头部，-n可以选择前n行查看
    tail -n filename 查看尾部，-n查看最后n行

top 查询当前进程，一般我用来看是否跑完了模型，之前想写一个邮件提醒的，但因为不会配置放弃了，还是回到了脑动预估时间、手动查看进程的原始人年代

ps -aux  列出全部进程
    a 全部进程 u 用户 ux 用户全部进程 

grep target filename 在文件中查询特定字段

sed 从文件中删除特定条件
    这里以我常用的一条命令为例：
    sed -i '/Warning/d' nohup.out 
    /d 是删除整行，因为nohup.out中经常带有大量的warning，但warning又不是error，我不在意，只想知道跑到哪一步出问题，如此处理就只剩下我关心的内容了。

nohup 不挂起
    因为目前模型普遍需要一个晚上的时间，nohup可以将任务后台不挂起解放我的笔记本回去打游戏！工具果然是摸鱼必备。

就介绍的这里，这些都只是我的业余爱好、辅助工具，打算用到啥学啥，得空再深入学习吧。
