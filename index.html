1无话可说与无话可说

Posted on 2020-07-15 | In 修炼爱情，我连起名都这么蠢笨了吗

船是怎么启航的？不知道，大概是风吧。
思绪是什么时候远航的？不知道，大概是夜幕降临之后吧。

突然觉得应该是有社科类的论文或者生物学的联合试验证明夜间的某些水平上升，更容易畅想吧。也说不定遐思是一种日光的PET探针，当夜幕降临，思绪就得以隐逸、放飞自己。

以往总觉得懒得解释理所当然，因为我也会对不重要的人疲于解释前后复杂的事情，但我想我对亲密关系的对象从来都很注意不说累到不想解释。思忖再三，这件事很复杂就简言之或者从头讲起，就算一遍问没解释，追问还是乐于讲完。想到这儿，我想唯一的解释就是我处处当心但还是有一些没注意到的细节，而人们一般更倾向于记忆令自己不喜的细节。
其实这里我就想解释我的标题了，我觉得无话可说存在两种含义，应当予以分别：第一种是态度上的欲言又止、无可奈何，第二种是纯粹的不知所措、脑袋放空。两种无话可说其实在争吵或者亲密关系的倦怠期里同时存在。啊呀，你在讲什么呀，是不是无话可说了，你猜对了还真是。

其实想到这儿我又开始乱扯开去，所谓安慰和随便说两句真的是送命题。或许在我实际体验中争吵可以跟面试或者报告的焦虑比肩，然而面试里我可以言之无物、报告中只需要回忆试验流程细节，争吵中却回到了最原始的白板状态。其实争执这件事挺像考试的，考前焦虑、蜜汁躁动（什么乱七八糟上厕所、肚子疼、口干舌燥想必各有各的经历），考中你绞尽脑汁想不出怎么构造等式二次求导那么你就是不会，哪怕你知道方向，考场上我如何念叨着要构造让我安慰时就如何念叨着讲点啥。
是啊，讲点啥呢？我知道啥啊，你让我现在随便讲点啥我也不知道讲啥，我好羡慕那些有东西可以扯的啊。我想过，是不是我不爱分享所以无话可讲，（虽然当年考的不是有话则长，无话则短，但我觉得这样的方式大家都舒服，何乐不为）但我也有对事情的分享欲、生活细节的赞美欲，我想我大概缺乏领导力，因为连随口扯淡也不会。这两天看了豆瓣（比微博好一些，但说实话有些言语还是逻辑诡异、难以理解），自我对照应当有幸能获得一个“小镇做题家”的勋章吧，虽然不甚荣誉。

女票（等我重新配置了hexo发出来的时候大概要加prefix了吧）告诉我觉得这样很冷漠，也是，所以真要问问题我也没有推辞过别人；告诉我要会哄会看眼色，实践下来看眼色也不是很有成就、哄就更不用说，眼色好看啊，氛围我也大概能懂了，说不生气是生气的诀窍也学会了，但情况变得更复杂了，如果说要安静到底有没有暗含“不生气”呢？（头疼，禁止套娃，禁止禁止套娃）至于哄，无话可说的标题还不够说明情况吗，所有看到的帖子都告诉我要哄，说好话，？？？（地铁，老头，手机.jpg）这不是脱裤子放屁吗，我也知道，这些情感博主能不能多一些documentation的作用，给个实例我才知道咋学啊。你瞧瞧，迟钝同学可不是实打实一个笨比，不配啊。

以前我总说要沟通，后来我觉得要有效沟通、高效沟通，但目前看来有什么说什么的坦诚确实很难即时实现，哪怕我们达成一致，但总会在第一次感到不适的时候回避问题。至于有效、高效的沟通，想来更难，因为讲一次是不够的，确实需要给予两三次机会告知更改，如果真的能够一次改变，那此子大有可为啊（突然想玩此子不可留的梗，也是，一说就能注意的人精也是有的，但绝大多数人都不是）。这里还牵扯到一个是否认为自己已经如实沟通的问题，在产生问题时认为自己已经讲的很清楚和认识到自己没有讲清楚但不能抽丝剥茧深入到根本问题的情况，前者的解决办法是学习怎么陈述，随便找一个帖子都有好为人师者教你，原则简单但实践不易，后者就已经很棒了，即使没能发现终极问题，但表现自己的真实情绪、杜绝再次发生不成问题。

很多时候也要讲究时间和场合，老生常谈了，但确实令人无奈，当一方正在形成自己的看法，那你的这些都可以封存起来了，讲不透是难受，不愿意讲是终极问题。和疑惑我是谁、我自哪儿来、我去到何处一样的等级，不愿意讲把刀子插在你心里，每想一次就转动一次，真痛苦。

无话可说的态度太挺伤人了，我对旁人无所谓，爱怎么表达怎么表达，教不会你就教不会你，与我何干？但真当刀子落在自己身上，那也确实委屈、无力。

爱意就像小学无聊的一边进水一边排水的数学问题，热恋正当时这点排量不被放在眼里，倦怠一至，你就能切身体验“病去如抽丝”的折磨。你只能用手去堵，进而用躯体，到后来，管口扩张到你塞不住了，池水放干到阈值了，那就再见吧。
看帖子甚至还去看如何挽留，有时候看着那些文字就挺累的，大概我的进水口还有一些流量，排水口也没有扩大多少吧。

在这期间，还有一个问题我在问自己，真的看起来只有我这么无辜幸苦的感动自己吗？我不知道，但我相信不是，love是两个人、四个字的事，在我的视角里只有我的努力很正常，但在看不到的地方对方也不时维护着这里的水量不降低呢。
我能确定吗？或许我的1/3是乐观主义者，我相信她。

这里一开始是打算只留给项目和技术的博客，这些文字也是我以往决不相信自己会为了情感而留下的。我想或许这就是人，多么单纯又复杂，如此决绝却又思虑繁多……再或者说一个人躯壳的年轻与沉暮的灵魂……

写在狗屁不通的文字前面

该作者试图平淡（寡淡）地开始讲述过往故事，但看起来没有什么故事感。
该文字试图走汪曾祺等的风格，但行文风格只有絮絮叨叨、精神病人式的落魄（压抑？）。
对了，还有这样（瞎补充/想到啥写啥，我好想说是天马行空的文字，算了，不配）
其内容试图罗贯中西，但只是流水账、前后狗屁不通、自我矛盾的文字，写这些的原因很可能（不，绝大多是因为突然/就很伤感）

**所有内容都属于胡言乱语，不具有指导作用，个人成见、偏见、短见诸君一笑而过。

为什么呢

我问你，一个人觉得只有自言自语（不对，只限于写出来）之后才能暂时缓解情绪是不是很蠢啊？
我觉得是欸，特别蠢。就算是有女票（或许热恋期无话不可讲，但那时什么都会被冲淡，到了倦怠期，用她的话说没那么爱了……不对，准备另外开一章在那儿讲的，），我原以为要一直走下去的相互支持也未必时刻能够体会。
如果蠢可以解决问题的话，我想我已经能发SCIENCE了。

开始了

什么算是材料？高中时期写些文字总是大杂揉，虽然也遵循先后的套路，但当时不觉得，看似混乱的结构总有自己的说法，文学大家可以肆意妄为，对于要得分的文章却不能率性而为。
btw，写到这儿怀念起初中时的一位同学，硕哥爱好杂文，受他的影响我当时最爱的文艺风标题是《写在<坟>的后边》，真是奇奇怪怪的年轻爱好呢！
btw2，写的时候我觉得不能老被情绪左右，所以边看边写，《虽然是神经病但没关系》，女主确实很漂亮。
好，回来。或许之后就是理科男了吧，虽然工作中没有什么高见（真一点点点点点微不足道的贡献），但套路却能很快熟识。我想自那起，我更愿意把自己的文字变成“报告文学”，杂七杂八，形似而非。看起来，这个人不像是有以文字传达心意的能力。
btw3，韩语的wuli表示我的/我们的，跟本地方言很近，只在第一个字的差异。

其实人真的会变的，外貌毫无疑问，但思维方式的改变才是一个人正真的变化，以前我觉得规则是世间一切的组成，掌握了数学法则、物理化学定律就能认识一切。科学原理上讲的通，一切自然都是可以解释的，但越深入就越能够体会科学的无止境。我们所有的定律都是基于特定条件下的，想要达到通解仍需努力。我想分子与蛋白质的结合总归会被微观研究所破解，抑制剂与非抑制剂的区别一定可以在结构上画出巧妙的一刀，但对于人的思维则没有这样的定律。尽管心理学已经总结归纳推理出了一系列人意识的特征或代表（这样的话脑机的原理我好像有一些赞同，以神经网络模拟神经网络，虽然后者数量众多没法以简单的少量网络囊括，那么dropout的提出真的是美妙的点子，既适合单纯的前者，又适合以简单模拟复杂，真有趣），但我仍然觉得由于变量茫茫而瞬息万变。
爱也是一样，伴侣所经历的才最会塑造人。窃以为很多校园情侣走不到最后很正常，因为并没有经历过“磨难”，这里只要是一起经历的重要事件、能记住的、有相互需要的。实际上，天南地北聚在一个学校的两个人很多时候是风花雪月度过校园时光，毕竟学校里能有什么大事。出了象牙塔，什么都会遇上，困难接踵而来，这才是现实，两个人才有一起度过的波澜，校园恋爱期间说感情淡了，没那么喜欢其实也稀疏平常。是啊，你想光怪陆离的人生故事总计也就40/2=20年（去除一半琐碎和不记事），平凡人的日常算半年吧，真的就开始需要创造不同了。
突然又想到，一开始炽热浓烈到达顶峰在我们这种情况下并不是什么好事，如果能一直见面维持自然淡的慢一些，而突发的疫情和迟迟不能返校，两边都不会也不能一块儿，那么只要言语或者情绪的一瞬间，堆积起来的雪山就会崩塌。人总是对变化敏感，会记住由不好到感觉还不错，也能记住最爱到好像没那么特别，太棒了，这样的机制对没有感情培养感情是最合适的，只需要寻找一个相处舒服的就可，然而对轰轰烈烈的，只能说遇到的时机是这么不妙，或许在这个情况下这样浓烈的契合更适合当朋友吧。我也挺羡慕677的勇敢奔跑，但人家毕竟多年情况差异太多。
对心理学的研究，hhh，仅限于看来两集b站的《变态心理学》，但我想，自然光在滤镜片后只显示单一的颜色，自然的，人在抱有一定看法的时候做什么都是错的，你知道这在心理学里叫什么效应吗？言语在我始终是苍白的（我，不包括那种大神，吉光片羽就能拨动心弦），或许立刻结束才体面一些，而结束不是最令人沮丧的，大概没有能超越不能被理解/被误解的了吧。而转换角度来看，在关系里的双方，一定都会有这样的时刻，无耐才是今晚的康桥。

人永远无法把自己拔高到四维观测自己，所谓的抽离出来审视自己往往还是会更多偏向关注。所以情感里会有自己是对的，别人是错的，你可以讲是自私，当然没问题，你要讲就要为了自己、亲密关系如果不能自己舒适那又为何要维护的话，也没有问题。我从来不care别人的想法和观点，总是纵容他们的想法、支持他们毫无道理的自我偏向，正常，人这么多年塑造的三观哪有那么容易改变？稍有一些融化就足以自慰。要变的太多了，双胞胎尚且有区分辨别，经历、想法能有绝大部分契合就已经是万幸，妄图凭借一句话或者一次争执改变？如果是法律规定一样，我相信可行。李家坡的鞭刑深入人心尚有这么多人犯禁。些微的改变确是不能如人意的，越是对比越是验证对方的先验结论，这也是没什么必要争辩的，哪有人时刻客观？为我自己想，符合我自己设想的发展路线，证明自己的想法、观念无误，我也不能免俗。（不是吹，不对，就是硬吹我自己，自认为很客观，每次二分法、换位思考查验，但也就是五十步笑百步吧，起码已经超越了99.9%人了，不敢说四个9，那也太硬吹了）

看，我这不就开始预测了吗！遇到一个一般的适合，经历一些事后会慢慢觉得越来越可。遇到炽热的适合，如果能够一起共度风雨，那是最好，真心祝愿他们。如果只是平平淡淡，那只能真心祝愿好运了，但经历了这么多也成熟了，生活也上了正轨，不是重大矛盾也就这样了。

还有很多时候，亲密关系里的一方会把周围环境的困顿归结于另一方，这个说法吧，说是也是，很多时候我们愿意往亲密关系的另一边倾泻情感，也倾泻不适，无论是否这些源自对方。这种说法不予置评吧。

人们总说要学会培养感情的能力、维护亲密关系的能力，因为是天平的两方难以达成势均力敌，说是尊重平等，总要有一个退步。很多时候这一方最好是男生，然而很多时候真的不会表达（无话可说又是一个我可以吐槽很久的东西了，里面的想法还不少，要转两圈），沟通（所有的沟通都要是形成文字的，或者说是形成面对面的，成文成语的，像法律文书一样表达，这可太难了）能让一切明了，但永远是说起来容易做起来难。不早了，明晚再写。

introduction-to-some-cmd-in-linux

Posted on 2020-07-15 | In 总结

因为实验室的webserver是Linux系统，而我日常使用win，初次接触还是有很多命令要学习，确实很令我头疼。

在这里，记录一下一个初学者的commad:

ls 列出当前目录下的文件 
    ls -l 是以列表形式的展示，简写： ll
    ls在windows的powershell中也支持，默认以列表展示（仅支持ls）

pwd 当前目录

cd target_path 与windows一致，切换路径

cat filename 查看文件的全部内容
    head -n filename 查看文件头部，-n可以选择前n行查看
    tail -n filename 查看尾部，-n查看最后n行

top 查询当前进程，一般我用来看是否跑完了模型，之前想写一个邮件提醒的，但因为不会配置放弃了，还是回到了脑动预估时间、手动查看进程的原始人年代

ps -aux  列出全部进程
    a 全部进程 u 用户 ux 用户全部进程 

grep target filename 在文件中查询特定字段

sed 从文件中删除特定条件
    这里以我常用的一条命令为例：
    sed -i '/Warning/d' nohup.out 
    /d 是删除整行，因为nohup.out中经常带有大量的warning，但warning又不是error，我不在意，只想知道跑到哪一步出问题，如此处理就只剩下我关心的内容了。

nohup 不挂起
    因为目前模型普遍需要一个晚上的时间，nohup可以将任务后台不挂起解放我的笔记本回去打游戏！工具果然是摸鱼必备。

就介绍的这里，这些都只是我的业余爱好、辅助工具，打算用到啥学啥，得空再深入学习吧。
